# Secret Messages

A simple web app to learn [Go](https://golang.org/project/).

It encrypts and decrypts messages using [AES](https://en.wikipedia.org/wiki/Advanced_Encryption_Standard).

Ready to deploy if you have an aws account.

```
eb init
eb create
```
